🔧 Troubleshooting

Script looks out of date after an update?

Your browser may be showing a cached version. A hard refresh forces it to reload the latest:

Windows / Linux: Ctrl + Shift + R
Mac: Cmd + Shift + R

Something not working as expected?

Try refreshing the page first — most minor glitches clear themselves. If you've unsaved work, use 💾 Save before refreshing. If the issue persists, drop us an email and we'll look into it.

If you downloaded ScriptWriter to use offline, you can get the latest version anytime by visiting rbeadle21.github.io and clicking ⬇ Download App again.

Actor Zone — Quick Guide

Get Your Script Ready

1
Upload your script — ideally a .playwriter.json file from the Script Editor. Word (.docx) and PDF files also work but may need tidying.
2
Select your character(s) from the list that appears, then click Confirm Character.
3
Choose your view — Full Script with your lines highlighted, My Scenes only, or Cue Script showing just your cue lines and your lines.
4
Use Self-Test Mode to hide your lines and click to reveal them one at a time — great for learning your lines.
⚡ Internet note: .json files work offline. Word and PDF files need internet the first time only — then they're cached.

How to Prepare a Script for the Actor Zone

Follow these steps to get the cleanest, most accurate script for your cast. This only needs doing once — then share the saved file.

1
Open ScriptWriter and go to the Script Editor tab
This is the main writing area — you should see the white page with block-style editing.
2
Import your script
Click 📥 Import in the top bar. You can drag in a Word (.docx) or PDF file, or load an existing ScriptWriter file. The script will be parsed into blocks automatically.
3
Check and fix the blocks
Scroll through the script. Each line should have the right type — Dialogue, Stage Direction, Act Heading etc. If anything looks wrong, hover over the block and click the ⇄ button to change its type. Character names should appear on the left of dialogue lines.
4
Check the Cast List
In the left sidebar, scroll down to Cast List. Make sure all character names are listed correctly. You can add or remove names there. This is what actors will see when they select their character.
5
Save the file
Click 💾 Save in the top bar. A save dialog will appear — save it somewhere easy to find. The file will be named something like your-play.playwriter.json.
6
Share the file with your cast
Send the .playwriter.json file to each actor — via email, WhatsApp, Google Drive, Dropbox, or any file sharing method. They open ScriptWriter, go to the Actor Zone tab, click Upload Script File, and load the file. No app to install, no account needed.
Why .json and not Word or PDF?
The .playwriter.json file preserves everything — character names, block types, cast list — exactly as you set them up. Word and PDF files have to be re-parsed each time, which can introduce errors with unusual formatting.
